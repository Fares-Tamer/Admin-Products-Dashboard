## Admin Products Dashboard

A frontend-only admin dashboard to manage products with full CRUD operations.

### Features
- Create / Update / Delete products
- Bulk create using count
- Search by title & category
- Sorting by price & total
- Dark / Light mode
- Authentication with Clerk
- Email verification required for creating products
- Persistent data using localStorage

### Tech Stack
- React
- Tailwind CSS
- Clerk
- React Toastify
- lucide react
